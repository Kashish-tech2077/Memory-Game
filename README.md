# 🧠 Memory Game — JavaScript Project

A fully responsive Memory Matching Game built using Vanilla JavaScript.
Flip cards, match emoji pairs, and complete the game in the least number of moves possible!

This project was built as part of my JavaScript learning journey and helped me understand DOM manipulation, game logic, conditional flows, and dynamically creating UI elements using JavaScript.

# 🧠 How The Game Works

1. A player clicks on a card to reveal its emoji.
2. The player clicks another card.
3. If both cards match:

   * The cards remain visible.
4. If both cards do NOT match:

   * The cards automatically hide again after a short delay.
5. The game continues until all pairs are matched.
6. After completing the game:

   * A popup displays total moves used.
   * Player can restart the game.

The lower the move count, the better the memory performance.
